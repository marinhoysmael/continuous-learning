# continuous-learning

Este repositório tem como propósito criar uma base de código de estudos em programação. Cada diretório representa uma linguagem de programação ou tecnologia específica, contendo exemplos, exercícios e projetos práticos para aprendizado contínuo.

## Estrutura

Cada pasta do repositório corresponde a uma linguagem ou tecnologia, com seus próprios exemplos e documentação:

- **kotlin/** - Estudos e exemplos em Kotlin
- _(outras linguagens/tecnologias serão adicionadas conforme o estudo avança)_

## Objetivo

Manter um registro organizado do aprendizado em diferentes tecnologias, facilitando consultas futuras e acompanhamento da evolução do conhecimento.
